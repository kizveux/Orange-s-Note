# Orange's Notes

Orange's Notes is a lightweight journaling web application built using HTML, CSS, and JavaScript. It enables users to create, edit, search, and export personal notes directly in the browser. All data is stored locally using the browser's `localStorage` API, making it fully client-side and installation-free.

## Features

- Create, edit, and delete notes
- Select date and mood for each entry
- Search notes by title or content
- Export individual notes to `.txt` format
- Clear all entries with confirmation
- Responsive layout for desktop and mobile
- Animated interface for enhanced user experience
